
# Trivia Game Sample

#### This is a sample react native app to highlight different aspects of React Native.

##### Features:
- Styled Components
- React Navigation
- React Native Reanimated and animatedSwitchNavigator
- Redux & Redux-thunks
- React-Native-Vector-Icons
- React Native Paper


###### Instructions: 
- `npm install`
- `react-native link`
- **For ios:** `cd ios` && `pod install`


Have fun!
